- Direct link to the GitHub repository:  
  - https://github.com/BxtGeek/BuildHypervisor  
- A clear call-to-action to support the project by giving it a ⭐ on GitHub  
- A **Buy me a coffee** support link  

## Deploy on GitHub Pages

1. Push this repository to GitHub  
2. In your repository, go to **Settings → Pages**  
3. Under **Build and deployment**, set:
   - **Source:** `Deploy from a branch`  
   - **Branch:** `main` (or your default branch)  
   - **Folder:** `/ (root)`  
4. Save and wait for GitHub to publish your site
